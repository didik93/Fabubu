# Fabubu 安装与故障排除指南

本指南将帮助您顺利安装 Fabubu 自动化发布助手，并解决在 macOS 上可能遇到的安全拦截问题。

## ⚠️ 重要提示：必须安装 Chrome 浏览器

**无论您使用的是 Windows 还是 macOS，本软件都依赖 Google Chrome 浏览器才能正常运行。**
请确保您的电脑上已安装最新版的 Chrome 浏览器。如果没有，请前往 [Google Chrome 官网](https://www.google.com/chrome/) 下载安装。
如果遇到运行问题（如点击打开按钮无反应），请尝试：

1. 更新 Chrome 到最新版本。
2. 清除浏览器历史缓存。

## 🍎 macOS 安装指南

### 1. 下载与安装

1. 下载最新版本的 `Fabubu_x.x.x_x64.dmg` (Intel芯片) 或 `Fabubu_x.x.x_aarch64.dmg` (M1/M2/M3芯片)。
2. 双击打开 `.dmg` 文件。
3. 将 **Fabubu** 图标拖入 **Applications (应用程序)** 文件夹。

### 2. 首次运行与常见报错 (⚠️ 必读)

#### 🛑 报错："Fabubu" 已损坏，无法打开

当您第一次尝试打开应用时，可能会看到如下图所示的报错，提示您**“将它移到废纸篓”**：

![安装报错](../ref/安装报错.png)

#### 🧐 原因

这不是因为文件真的损坏了！
这是 macOS 的 **Gatekeeper 安全机制**。因为 Fabubu 是个人开发者项目，没有购买苹果昂贵的开发者签名证书进行“公证 (Notarize)”，macOS 会默认拦截从网络下载的此类应用，并强制打上“隔离 (Quarantine)”标记。

#### ✅ 解决方法

1. 打开 **终端 (Terminal)** (可以在“启动台” -> “其他”中找到，或按 `Cmd + Space` 搜索 "Terminal")。
2. 复制以下命令并粘贴到终端中，然后按 **回车**：

   ```bash
   sudo xattr -r -d com.apple.quarantine /Applications/Fabubu.app
   ```

3. 终端会提示您输入开机密码（输入时屏幕上不会显示字符，输完直接按回车即可）。
4. 完成后，重新点击 Fabubu 图标即可正常打开。

## 🪟 Windows 安装指南

### 1. 下载与安装

1. 下载 `Fabubu_x.x.x_x64-setup.exe`。
2. 双击运行安装程序。
3. 可能会弹出 **Windows SmartScreen** 提示“已保护你的电脑”。
   - 原因同上，是因为没有购买微软的代码签名证书。
   - 点击 **“更多信息”** -> **“仍要运行”**。

### 2. 安装完成

- 安装完成后，Fabubu 会自动启动。
- 程序会自动检测并配置运行环境。
